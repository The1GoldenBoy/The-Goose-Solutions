## 🪿 Goose Command Center — le Super App Master Plan, livré

Implémentation complète du master plan Obsidian (`docs/MASTER-PLAN.md`) : un OS multi-agents avec les 5 vues (Office, Factory, Meeting Room, Completion Watchdog, Mémoire), le cycle spec → exécution → preuve → validation → archive, le watchdog anti-stale avec escalade à Max, et le failover directeur Goose → Maverick.

```bash
cd command-center
npm start    # dashboard → http://127.0.0.1:7777
npm test     # 9 tests verts
npm run demo # cycle complet en 8 étapes
```
